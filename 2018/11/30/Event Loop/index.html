Event Loop 是什么？

为啥需要 Event Loop

  • 如图当js主线程运行时就会产生堆（heap）和栈（stack）。
  • js在执行的过程中会产生执行环境,当遇到异步事件时就会把它放到任务队列中（如执行栈中的去调用一些webApi，click事件等被放在任务队列中）。
  • 当执行栈中执行完毕后，主线程就会去读取任务队列把第一个放到执行栈执行（图中执行栈执行完会把任务队列中的click事件的回调放到执行栈执行）
  • 当执行栈执行时遇到异步，又回放到任务队列中
  • 执行栈清空后，会去检查任务队列中是否有需要执行的，如果有就拿过来执行，会一直重复这个动作，称为Event Loop
